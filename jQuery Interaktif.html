FAQ Interaktif

Apa itu jQuery?
jQuery adalah pustaka JavaScript yang memudahkan manipulasi DOM, event handling, dan animasi.
Bagaimana cara menginstall jQuery?
Anda bisa mengunduhnya dari situs resmi atau menggunakan CDN seperti Google atau jQuery CDN.
Apa fungsi .click() di jQuery?
.click() digunakan untuk menangani event klik pada elemen HTML.
Apa perbedaan .hover() dan .toggle()?
.hover() menangani mouseenter dan mouseleave, sedangkan .toggle() menampilkan atau menyembunyikan elemen.
Mengapa menggunakan jQuery untuk FAQ?
jQuery memudahkan pembuatan interaktivitas seperti toggle jawaban tanpa banyak kode JavaScript murni.
